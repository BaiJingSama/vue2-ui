# vue2-ui

## 介绍

> 这是我在工作后的日常时间写的一个Vue的ui框架，希望对你有用

## 开始使用

1. css样式


使用本框架前请在css中开启 border-box

```
*,*::before,*::after{box-sizing: border-box}
```
你还需要设置默认颜色等scss变量
```
$font-size: 14px;
$button-height: 32px;
$button-bg: white;
$button-active-bg: #eee;
$border-radius:4px;
$color: #999;
$border-color: #999;
$border-color-hover: #666;
```
2. 安装

npm安装

```
npm i vue-starry-sky-ui
```

yarn安装
```
yarn add vue-starry-sky-ui
```
